首先明确我要做的是一个俄罗斯方块：
主要由两个部分组成一个是方块，另一个就是游戏画板
1：方块
用来方块的生成，移动翻转固定，主要是用到了矩阵，1代表方块存在0代表不存在通过矩阵可以
画出基本的方块了
2：游戏画板
游戏画板用于画表格横线竖线，并加上定时器和监听器，用于反馈用户按键，传递到方块工厂来控制方
块，并且检查方块是否碰撞或消行等状态
主要的思想就是，给一个方块集合，判断在顶部还是底部，顶部代表结束，输出分数就好了，在底部则
看底部是不是铺满的，若是，则消解这一行，加分即可，用户可以在操作界面上进行操作，如停止，方
块的翻转等等内容


实现的主要功能：
1：方块反转、左移、右移、下降
2：定时器定一特定时间作为时间间隔触发使方块定时下落一格
3：判断是否碰撞（包括与围墙，底部，已固定方块），碰撞则停止并生成新的方块下落
4：判断是否能消行，且消行要加分记录并显示在界面上
5：生成方块前检查最顶行有没有方块，有则弹框提示结束游戏并显示所得分数。
6：暂停游戏
7：难度的变换
 

代码分为三个主要的内容
1:Block类
主要是方块的基本方法如方块固定、新建、翻转、左移、右移、下移
在构造方法有newblock,add,right,down,left,turnstate分别对应上面的功能
2：GamePanel类
即游戏界面的相关操作
分别进行了游戏面板的行数列数的画，以及内部timer的触发，对于游戏过程中的计分，消去已经满的一
行方块，在里面我调用了block类，下面构造方法有uplevel，downlevel,cleanmap,drawwall等分别对应
的是增加难度减小难度，清空界面等功能，接下来再画操作界面，即用户如何通过按键来操作，画横线和竖
线，就可以构造操作界面的面板了，后把各个功能键引入即可。接下来就是遍历方块，判断游戏是否结束，
并且记录相关的分数
3：RussiaBlocks类
也是游戏的入口：对于游戏名称的引入，添加游戏画板，是否进行游戏，以及timer的开始运行，以及主类的
引入
